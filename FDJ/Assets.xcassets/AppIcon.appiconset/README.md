# App Icon Requirements

This app icon needs the following images:

- **1024x1024** PNG files for:
  - Light mode icon
  - Dark mode variant (optional)
  - Tinted variant (optional)

## Design Guidelines

- Use a sports/court/gambling theme
- The app name is "Sports Odds Tracker"
- Primary color: Blue (#007AFF)
- Consider using a basketball court or ticket icon
- Keep it simple and recognizable at small sizes

## Adding Icons

To add your app icons:
1. Create or obtain a 1024x1024 PNG image
2. Add it to this folder with the following naming convention:
   - `icon.png` - Standard icon
   - `icon-dark.png` - Dark mode variant
   - `icon-tinted.png` - Tinted variant

3. Update the Contents.json file to reference your images
